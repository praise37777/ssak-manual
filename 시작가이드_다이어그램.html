3
세부 실행 방법
1
💻 Claude 새 채팅을 열고 Sonnet 모델을 선택합니다
claude.ai 접속 → 왼쪽 위 "New chat" 클릭 → 상단에서 모델을 Sonnet으로 선택해주세요.
▼
2
📗 Claude에게 이 메시지를 복사해서 보냅니다
아래 메시지를 그대로 복사해서 Claude 채팅창에 붙여넣으세요.
[사이트 주소] 부분만 오늘 작업할 사이트 주소로 바꿔주세요.
Claude에게 보낼 첫 메시지
안녕! 싹싹컴퍼니 워드프레스 자동화 작업을 할 거야. 아래 URL의 파일을 먼저 읽고 나를 도와줘: https://raw.githubusercontent.com/praise37777/ssak-manual/main/sonnet_context.md 오늘 작업할 사이트: [여기에 사이트 주소 입력] 파일 다 읽었으면 어느 단계부터 시작하면 되는지 알려줘.
▼
3
📘 새 탭에서 매뉴얼을 엽니다
브라우저 새 탭에서 1번 매뉴얼 URL을 열어요. 이 매뉴얼은 10단계로 나뉘어 있어요.
왼쪽에 목차가 있으니 1단계부터 순서대로 따라 하세요.
▼
4
📸 각 단계 끝나면 스크린샷 찍어서 Claude에게 보고
단계 하나 끝날 때마다 결과 화면을 스크린샷 찍어서 Claude 채팅창에 올려요.
Claude가 확인해주고 다음 단계로 가라고 말해줘요.
문제가 있으면 바로 알려줘요.
4
꼭 지켜야 할 것 / 절대 하면 안 되는 것
✅ 꼭 지켜야 할 것
  • 매 단계마다 스크린샷 찍어서 Claude에게 확인받기
  • 매뉴얼 순서대로 1단계부터 진행하기
  • 헷갈리면 무조건 Claude에게 먼저 물어보기
  • 이상한 오류나 화면이 나오면 스크린샷과 함께 질문하기
  • 작업 중 모르는 게 있으면 바로 대표님께 연락
🚫 절대 하면 안 되는 것
  • 원본 워크플로 / 시나리오 수정 금지 (꼭 copy 버전만!)
  • wp-config.php 같은 파일 마음대로 수정 금지
  • 비밀번호나 API Key를 그대로 채팅창에 붙여넣기 금지
  • 모르는 플러그인 함부로 설치 금지
  • 단계 건너뛰기 금지 (순서대로 진행!)
🎯
